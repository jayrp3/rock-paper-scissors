Rock, Paper, Scissors (RPS)
=====================
Objective: 
Runs through 5 rounds of Rock, Paper, Scissors (RPS) against
a Computer (CPU) and outputs a winner with a final score.

Functions:
- playGame()
# Starts 5 rounds of RPS against a CPU
- playRound()
# Plays one round of RPS against a CPU 
# Outputs a winner
- getHumanChoice()
# Asks user for input for a given round
- getComputerChoice()
# Generates a CPU choice for a given round
